# HarjoitusKaksitikape

Harjoitustyö kaksi kurssille Tietokantojen perusteet. Lähdekoodi on nähtävissä tällä github sivustolla, ja netissä toimiva sivu pohjautuen tähän koodiin osoitteessa https://gentle-temple-58818.herokuapp.com/

Toiminnallisuutta on, että voi selata kaikkia tallennettuja kysymyksiä painamalla "kysymyksiin" ja tallennetut vastaukset painamalla "vastauksiin".
Painamalla "Vastauksiin" pääset sivulle, jossa on näkyvillä kysymysteksti ja listattu kaikki kysymykseen liitetyt vastaukset.
Molemilla sivuilla jokainen tallennettu tieto voidaan painamalla "Poista" kyseisen kysymyksen kohdalla. Huomioi, että poistamalla kysymys, kaikki siihen viittaavat vastauksetkin poistetaan automaattisesti.
Uusi kysymys voidaan lisätä täyttämällä tekstikentät, ja painamalla "Uusi kysymys/vastaus". Jättämällä tekstikenttä tyhjäksi, tallentaa vain tyhjän rivin. 
Jos vastaukseen ei anna kysymyksen, johon vastataan, id:tä, ei vastaus tallennu.
